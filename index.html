Art 372 Portfolio

Hello! My name is Noah Turcotte, I'm am a student at West Virginia University majoring in Game Design and Interactive Media. This page serves as a gallery of my work for Art 372 - Interactive Design. Each project completed in this course is accessable from this webpage by clicking on the images below. Hover over each image to get a description of each project.

To understand the basics of webdesign, a simple webpage feature a story that followed the hero's journey was constructed. This particular narrative revolves around a package being delivered to the wrong address.

Cartoon Characters were put through the wringer, as trashy clickbate articles meaning to soil their name were written about them. The cast of chowder is exposed in these headlines, the only way to figure out the truth is to click this image!

Whack a What? Using html,css code, and javascript, a simple arcade style whack a mole game was developed. In this game, the player is tasked with hitting as many pinatas as they can before time runs out. Watch out for the bee hives!

Using Jquery, an interactive webpage was created exploring a form of analog technology. Watch the Rockem Sockem Robots fight while learning some facts about the game in the process.

This project was created in conjuction with ChatGPT, the task was to have an ai chat bot create a canvas based game while a design was created integrating the canvas into a visually interesting webpage.
